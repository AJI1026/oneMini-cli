# 提示词润色

## 流程

1. 识别目标介质：静帧 / 视频 / 角色立绘 / 场景 / 海报
2. 提取主体、动作、环境、光影、镜头、风格、负面要素
3. 输出「短版」与「完整版」两档；完整版分段清晰，便于复制到创作页
4. 若用户提供参考图说明，补上构图与一致性约束（服装、脸型、色调）

## 输出格式

```markdown
## 意图确认
（介质、画幅、风格关键词）

## 短版提示词
（一行，适合快速试图）

## 完整版提示词
- 主体：
- 场景：
- 光影与色彩：
- 镜头与构图：
- 风格与质感：
- 约束：

## 负面提示（可选）
…
```

## 原则

- 先结构后辞藻；避免无意义的堆砌词
- 中英混排时，关键专有名词保持一致
- 视频提示补充运动与时长节奏，不写无法生成的镜头特技
