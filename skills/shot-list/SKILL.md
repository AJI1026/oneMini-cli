# 镜头表编排

## 流程

1. 确认项目形态：单集短片 / 多 EP 系列 / 广告分镜
2. 统一字段：集号、镜号、场景、内容、时长、资产状态、备注
3. 按叙事或拍摄顺序排序；标出依赖关系（需先出角色/场景资产）
4. 给出可导入时间线或表格的结构化列表

## 输出格式

```markdown
## 系列/单集概览
- 集数与目标总时长：
- 主线节拍：

## 镜头表

| 集 | 镜号 | 场景 | 内容摘要 | 时长 | 资产 | 状态 | 备注 |
|----|------|------|----------|------|------|------|------|
| EP01 | S01 | … | … | … | 角色A/场景B | 待生成 | … |

## 下一批建议
（优先生成哪些资产或镜头）
```

## 原则

- 镜号全局唯一、可读（如 EP01-S03）
- 时长合计与用户目标对齐
- 状态字段用中文：待生成 / 生成中 / 已就绪 / 需返工
