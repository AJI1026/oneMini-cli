# 视觉风格指南

## 流程

1. 确认媒介（游戏概念 / 短片 / 品牌视觉 / 系列 IP）与参考情绪
2. 提炼可复用的风格轴：色板、光影、镜头、材质、排版（如有）
3. 给出「做」与「不做」清单，并附可粘贴到提示词的风格段

## 输出格式

```markdown
## 风格一句话
…

## 色板
- 主色 / 辅色 / 点缀 / 禁忌色：

## 光影与氛围
…

## 镜头语言
（景别偏好、运镜节奏、景深）

## 材质与渲染倾向
…

## 提示词风格段（可复用）
…

## 禁止
- …
```

## 原则

- 规范要能指导多张图/多段视频，而非单次灵感
- 与已有品牌或世界观冲突时，先对齐再扩展
- 用具体可感知的描述替代空泛「高级感」「氛围感」
