# 角色设定卡

## 流程

1. 确认用途：出图参考 / 剧情 NPC / Lore 入库 / 多角色对照
2. 收集已知信息；缺失项用简短假设补齐并标注「待确认」
3. 输出结构化设定卡，并附 1–2 条可直接用于出图的提示词

## 输出格式

```markdown
## 基本信息
- 姓名 / 称呼：
- 年龄观感 / 身份：
- 一句话人设：

## 外形
- 体型与五官：
- 发型发色：
- 标志性特征：

## 服装与道具
- 日常装：
- 战斗/正装（如有）：
- 随身道具：

## 性格与关系
- 性格关键词（3–5 个）：
- 说话口吻：
- 与关键角色关系：

## 出图提示词
- 正面立绘：…
- 半身/场景：…
```

## 原则

- 外形描述要可绘制，避免仅写性格形容词
- 与用户已有世界观冲突时，优先沿用已有设定并指出差异
- 多角色时保持风格与时代设定一致
